## 📖 О проекте

**MusicPlatforme** — это бесплатная веб-платформа для прослушивания музыки, созданная и поддерживаемая **[VriskasYT](https://github.com/vriskasYT)**. Сайт предоставляет полноценный музыкальный сервис прямо в браузере — без рекламы, без подписок, без ограничений.

MusicPlatforme совмещает в себе онлайн-плеер, каталог музыки, персональные рекомендации, пользовательские коллекции, загрузку собственных треков и дополнительные функции для более удобного и современного прослушивания музыки.

> **Слоган:** *MusicPlatforme — Слушай музыку бесплатно*

| ▶️ **Встроенный музыкальный плеер** | Полноценное воспроизведение музыки прямо в интерфейсе сайта |

| ⏩ **Перемотка трека** | Управление позицией воспроизведения внутри композиции |

## 🚀 Быстрый старт

Просто откройте сайт в браузере:

```
https://vriskasYT.github.io/musicplatforme
```

Или запустите локально:

```bash
# Клонируйте репозиторий
git clone https://github.com/vriskasYT/musicplatforme.git

# Перейдите в папку
cd musicplatforme

# Откройте index.html в браузере
# Или используйте локальный сервер:
npx serve .
# или
python -m http.server 8000
```

## 🌟 Почему MusicPlatforme — это больше, чем просто плеер

MusicPlatforme — это не просто страница с музыкой, а цельная музыкальная экосистема в браузере. Здесь можно:

- находить новые треки;
- следить за артистами;
- собирать любимую музыку в одном месте;
- создавать собственные плейлисты;
- загружать свои треки;
- использовать дополнительные функции плеера;
- настраивать прослушивание под себя;
- продолжать прослушивание более удобным способом, в том числе на другом устройстве.
