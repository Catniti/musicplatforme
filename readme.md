## 📖 О проекте

**MusicPlatforme** — это бесплатная веб-платформа для прослушивания музыки, созданная и поддерживаемая **[VriskasYT](https://github.com/vriskasYT)**. Сайт предоставляет полноценный музыкальный сервис прямо в браузере — без рекламы, без подписок, без ограничений.

> **Слоган:** *MusicPlatforme — Слушай музыку бесплатно*

---

## ✨ Возможности

| Функция | Описание |
|---|---|
| 🔥 **Новые треки** | Лента свежих загрузок от пользователей |
| 🏆 **Популярное** | Самые прослушиваемые треки на платформе |
| ⭐ **Рекомендации** | Персональные рекомендации сервиса |
| 🎤 **Артисты** | Каталог артистов с возможностью подписки |
| 🌊 **Моя волна** | Персональная радиостанция, которая подстраивается под ваши вкусы |
| 📻 **Радио России** | Встроенный каталог российских радиостанций с запасными потоками и fallback-механизмами |
| 📊 **Чарты** | Самые популярные треки прямо сейчас |
| 🆕 **Новинки** | Свежие релизы за последние 7 дней |
| ❤️ **Любимое** | Сохраняйте понравившиеся треки в личную коллекцию |
| 🎵 **Плейлисты** | Создавайте и управляйте собственными плейлистами |
| 📜 **История** | Полная история прослушивания |
| 👥 **Подписки** | Подписывайтесь на артистов и следите за их новыми релизами |
| ⬆️ **Загрузка треков** | Загружайте собственную музыку на платформу |
| 🔍 **Поиск** | Быстрый поиск по трекам и артистам |
| 👤 **Профиль** | Авторизация и личный кабинет пользователя |

## 🚀 Быстрый старт

Просто откройте сайт в браузере:

```
https://vriskasYT.github.io/musicplatforme
```

Или запустите локально:

```bash
# Клонируйте репозиторий
git clone https://github.com/vriskasYT/musicplatforme.git

# Перейдите в папку
cd musicplatforme

# Откройте index.html в браузере
# Или используйте локальный сервер:
npx serve .
# или
python -m http.server 8000
```
