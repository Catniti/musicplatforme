## 📖 О проекте

**MusicPlatforme** — бесплатная веб-платформа для прослушивания музыки, созданная и поддерживаемая **[VriskasYT](https://github.com/vriskasYT)**. Сайт предоставляет полноценный музыкальный сервис прямо в браузере — без рекламы, без подписок, без ограничений.

> **Слоган:** *MusicPlatforme — Слушай музыку бесплатно*

---

## ✨ Возможности

| Функция | Описание |
|---|---|
| 🔥 **Новые треки** | Лента свежих загрузок от пользователей платформы |
| 🏆 **Популярное** | Самые прослушиваемые треки на сайте |
| ⭐ **Рекомендации** | Персонально подобранная музыка на основе интересов пользователя |
| 🎤 **Артисты** | Каталог артистов с возможностью перехода в их профиль и подписки |
| ❤️ **Любимое** | Сохраняйте понравившиеся треки в личную коллекцию |
| 🎵 **Плейлисты** | Создание, хранение и управление собственными плейлистами |
| 📜 **История прослушивания** | Полная история недавно прослушанных треков |
| 👥 **Подписки** | Подписка на артистов и отслеживание их новых релизов |
| ⬆️ **Загрузка треков** | Возможность публиковать собственную музыку на платформе |
| 🔍 **Поиск** | Быстрый поиск по трекам, артистам и музыкальному контенту |
| 👤 **Профиль** | Авторизация и личный кабинет пользователя |
| ▶️ **Встроенный музыкальный плеер** | Полноценное воспроизведение музыки прямо в интерфейсе сайта |
| ⏯ **Управление воспроизведением** | Пауза, продолжение, переключение между треками и базовые элементы управления |
| ⏩ **Перемотка трека** | Управление позицией воспроизведения внутри композиции |
| 🔊 **Регулировка громкости** | Быстрое изменение уровня громкости во время прослушивания |
| 🎚 **Музыкальные эффекты** | Дополнительные аудионастройки и эффекты для более гибкого звучания |
| 📱 **Прослушивание на другом устройстве** | Возможность перенести или продолжить воспроизведение на другом устройстве |
| 📻 **Радио России** | Встроенный каталог российских радиостанций |
| 📊 **Чарты** | Самые популярные треки прямо сейчас |
| 🆕 **Новинки** | Свежие релизы за последние 7 дней |
| 🛡 **Авторизация** | Вход через Google, GitHub, Яндекс и Email + Yandex SmartCaptcha |
| 🪄 **Удобная SPA-навигация** | Быстрое перемещение между разделами без перезагрузки страницы |
| 🧠 **ИИ-модерация** | Автоматическая проверка контента на нежелательный материал |
| 🛡 **Верификация артистов** | Система подтверждённых артистов |

## 🛠 Технологии

- **HTML5 / CSS3 / JavaScript** — Single Page Application (SPA)
- **GitHub Pages** — хостинг и деплой
- **Google Fonts (Inter)** + **Material Symbols** — типографика и иконки
- **Google Firebase** (Auth + Realtime Database) — бекэнд и авторизация
- **Yandex SmartCaptcha** — защита форм от ботов
- **Material You** — дизайн-система

## 🚀 Быстрый старт

Просто откройте сайт в браузере:

```
musis.ru
```

Или запустите локально:

```bash
git clone https://github.com/vriskasYT/musicplatforme.git
cd musicplatforme
npx serve .
# или
python -m http.server 8000
```
